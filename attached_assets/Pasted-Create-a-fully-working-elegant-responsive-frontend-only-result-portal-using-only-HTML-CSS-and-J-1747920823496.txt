### ✨ Design & Styling Instructions

- **Modern government-style design** inspired by CBSE result portals
- Color Palette:
  - Background: `#f0f4ff` or `#e6f0fa`
  - Card: White with `box-shadow: 0 0 10px rgba(0,0,0,0.1)`
  - Header: `#003366` or `#1b3a57`
  - Accent colors: blue shades, soft borders, professional font (`Segoe UI`, `Roboto`, `sans-serif`)
- Rounded corners, clean form spacing, hover and focus effects
- Responsive via media queries
- **No disclaimers** — footer should simply say:  
  `Designed and Developed by Adarsh Dubey.` (centered)

### 🖨️ Watermark for Printed Page Only

- When the user prints the result:
  - Show a semi-transparent, large **watermark across the page**:
    - Text: **"VISHAL MEGA MART™"**
    - Styling:
      - Font size: large (e.g., `5rem`)
      - Color: light grey or `rgba(0,0,0,0.1)`
      - Rotate the text diagonally (`transform: rotate(-30deg)`)
      - Fixed position in center
- Ensure watermark is:
  - **Only visible in print mode**
  - **Not visible on screen view**

Use CSS `@media print { ... }` to handle this.
